add_goal_object(specific_object, x, y, nb)

This function sets a specified quantity of a given object as a goal at a certain location. By "object" we mean a type of object that can be taken or put down by Reeborg.

Cette fonction spécifie la quantité d'un certain type d'objet qui doit être mis comme but à un endroit donné. Par "objet", on entend ici un objet qui peut être transporté ou déposé par Reeborg.

add_new_object_type(specific_object, url, url_goal)

This function makes it possible to add new objects. If the name of an existing object is specified again, the images for that object are replaced. Two images must be provided: one for the object itself, and another when this object is specified as a goal. N.B. existing names are the English ones.

Cette fonction permet l'ajout de nouveaux objets. Si le nom d'un objet existant est spécifié, les images pour cet objet seront remplacées par les nouvelles images fournies. N.B. les noms d'objets par défaut sont des noms anglais (par exemple "token" plutôt que "jeton").

add_tile_type(tile)

This function makes it possible to add new tiles. If the name of an existing tile is specified again, it is replaced by a new one which may have completely different characteristics.

Cette fonction permet l'ajout de nouveaux tuiles. Si le nom d'une tuile existante est spécifiée, elle est remplacée par la nouvelle qui pourrait avoir des caractéristiques complètement différentes. N.B. les noms de tuiles par défaut sont des noms anglais (par exemple "grass" plutôt que "gazon").

Parameters:
Name Type Description
tile Object

A Javascript object (similar to a Python dict) that describes the properties of the tile.
Un objet javascript (similaire à un dict en Python) qui décrit les propriétés de la tuile.

Properties
Name Type Argument Default Description
name string

The unique name to be given to the tile.
Le nom unique donné à la tuile
public_name string <optional>

If various tiles are meant to represent the same type of tile (e.g. different shades of "grass") this attribute can be used to specify that single name.
url string <optional>

If a single image is used, this indicated the source. Either tile.url or tile.images must be specified.
images Array.<string> <optional>

If multiple images are used (for animated tiles), this contains the various URLs. Either tile.url or tile.images must be specified.
selection_method string <optional>
 "random"

For animated tiles; choose one of "sync", "ordered" or "random"
