Brython version

To write Python scripts, all you have to do is :

  • include the line <script src="brython.js"></script> in the HEAD section of your HTML file ; if the file brython.js in not in the same folder as the HTML page, replace "brython.js" by a relative or absolute path pointing to brython.js
  • run the function brython() on page load : <body onload="brython()">. You can also call it with a debug mode as argument : <body onload="brython(1)"> : this will write error messages to the browser console
  • write Python code in scripts inside the page :
    <script type="text/python">
(... python code ...)
</script>
Visit the Brython site for a gallery of examples, documentation and contacts if you need help

Have fun !
