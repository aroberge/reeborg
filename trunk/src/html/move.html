First program

In the third panel, notice a drop-down list with Select a world written. Click on it and select "Alone"; you will see Reeborg appear in the world, in the bottom left corner, as well as some buttons on the top.

Click on "Execute" and watch what happens.

Try it!

Congratulations!

You had Reeborg take its first steps. In the center panel, there is a single instruction:

move();

That is the command that Reeborg executed. Notice how the "Execute" button is now disabled; click on "Reload" to enable it again, and try to run it again. By the way, a synonym for Execute is Run (as in, run a program). When you are ready, move to the next lesson.
