단계적 정제: 3부
===================

알아채셨겠지만, 프로그램은 작동을 하지 않는다. 
리보그 주변에 벽이 없는 상태에서 리보그가 무한 반복상태에 놓여지게 된다. 
아래와 같이 오른쪽을 회전을 한 후에 리보그가 ``move()`` 명령문으로 한 칸 앞으로 이동할 필요가 있다::

    put()
    move()
    while not object_here():
        if right_is_clear():
            turn_right()
            move()
        elif front_is_clear():
            move()
        else:
            turn_left()
